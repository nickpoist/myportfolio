Brain Surgery Simulation Robot

The main construction of this project is a 3-DOF robot that is controlled by Matlab code, an Arduino,and three motors. On the back of the robot, a camera is mounted that takes an image of whatever is in front of the robot. The camera acts as the eyes of the robot. Likewise, the Matlab code acts as the brain of the robot. Once the image from the camera is sent to Matlab, it is able to process the image and detect the fiducials. Once it knows the location of the fiducials, the motors act as the arms of the robot and moves the needle to the location identified by the Matlab code and then returns to its original position.

Capstone Design Project

The Shooter Tracker was a semester long project of a basketball return system. The system operated with a Raspberry Pi in conjuction with and Arduino Uno. The system used HSV Color segmentation to analyze a users position on the court and return a ball to them.
